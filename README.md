# Hack4Health - Alzheimer's Disease Project

This project focuses on EEG data analysis for Alzheimer's disease research.

## Project Structure

```
├── src/
│   └── eeg/
│       ├── features.py      # Feature extraction
│       ├── load_data.py     # Data loading utilities
│       ├── pipeline.py      # Processing pipeline
│       └── preprocess.py    # Data preprocessing
├── requirements.txt         # Python dependencies
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Dependencies

- mne
- numpy
- scipy
- pandas
- scikit-learn
- antropy
